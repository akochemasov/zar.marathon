## Development

```shell
yarn run start
```

Проверка кода

```
yarn eslint .src/
```

Форматирование кода

```
yarn pretty-quick
```

## Build

Build dev

```shell
yarn run build:dev
```

Build prod

```shell
yarn run build:prod
```
